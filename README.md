# Profit-Analysis
# PROFIT ANALYSIS USING MULTIPLE LINEAR REGRESSION AND POWER BI

## Project Overview

This project analyzes the factors affecting startup profitability using Multiple Linear Regression and Business Intelligence techniques. The dataset contains information on 50 startups from New York, California, and Florida, including expenditures on Research & Development (R&D), Administration, and Marketing, along with corresponding Profit values.

The objective of the project is to identify the relationship between startup expenditures and profitability, develop a predictive model for profit estimation, and create interactive visualizations using Power BI.

---

## Tools and Technologies Used

* Microsoft Excel
* Multiple Linear Regression
* Power BI
* MySQL Database

---

## Dataset Features

The dataset contains the following variables:

* R&D Spend
* Administration Spend
* Marketing Spend
* State
* Profit

---

## Methodology

1. Data Extraction from MySQL Database
2. Data Cleaning and Validation
3. Exploratory Data Analysis
4. Multiple Linear Regression Model Development
5. Model Evaluation using R² and Adjusted R²
6. Profit Prediction
7. Power BI Dashboard Development

---

## Regression Model

Profit = 50122.19 + 0.8057(R&D Spend) − 0.0268(Administration Spend) + 0.0272(Marketing Spend)

---

## Model Performance

* R² = 0.9507
* Adjusted R² = 0.9475
* Significance F = 4.53 × 10⁻³⁰

The model explains approximately 95% of the variation in profit and demonstrates excellent predictive performance.

## Key Findings

* R&D Spend has the strongest positive impact on profit.
* Marketing Spend positively influences profitability.
* Administration Spend has minimal impact on profit.
* California startups generated the highest overall profit.
* The regression model can effectively predict future profits.
